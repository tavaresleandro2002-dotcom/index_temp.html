# Gestão e Automação de Usuários

## Descrição
Automatização e controle de usuários em Active Directory, com suporte integrado via Excel e relatórios de inventário de máquinas.  
O projeto demonstra organização, automação de processos e monitoramento de usuários e equipamentos corporativos.

## Ferramentas
- Active Directory
- Excel
- TOTVS RM

## Objetivo
Mostrar habilidades em:
- Criação e gerenciamento de usuários e grupos
- Controle de inventário via planilhas
- Análise de relatórios
- Integração de dados entre sistemas
